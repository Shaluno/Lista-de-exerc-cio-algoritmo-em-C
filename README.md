# Lista-de-exerc-cio-algoritmo-em-C
Trabalho acadêmico com foco em desenvolvimento básico e prática na linguagem C.

Exercícios em C - Estruturas de Controle

Este repositório contém uma coleção de exercícios desenvolvidos em linguagem C, com foco no aprendizado das principais estruturas de controle, como:

Estruturas de repetição (while, do...while)
Estruturas condicionais (if, else, switch case)
Validação de dados
Lógica de programação básica

Objetivo:

O objetivo deste projeto é praticar conceitos fundamentais de programação, resolvendo problemas simples e comuns em atividades acadêmicas.

Conteúdo:

Os exercícios abordam temas como:

Verificação de números positivos
Tabuada utilizando repetição
Números primos
Contagem de números ímpares
Soma de números pares
Contagem de dígitos
Menus interativos
Validação de entrada do usuário
Uso de switch case em situações reais
Simulações (calculadora, caixa eletrônico, robô, etc.).
